Rock-Paper-Scissor

Welcome to the game feild! To start the game click start button.

Rules:

Here, you are playing against the computer.

1. Rock beats Scissors.
Rock crushes Scissors.

2. Scissors beats Paper.
Scissors cuts Paper.

3. Paper beats Rock.
Paper covers Rock.

4. Tie.
If both players choose the same option (e.g, both choose Rock), the game is tie, and no one wins.
